# Soil Moisture Sensor (yl69)

## Hardware

-Sensor: YL69 (soil moisture probe) + 10kΩ resistor (voltage divider)

-Microcontroller: Arduino UNO (ATmega328P)

-Pins Used:

A0 → Soil probe analog sense (divider output)

D7 → Soil probe power control (probe powered only during reading)

D2 → DHT11 data pin

5V → DHT11 VCC

GND → Common ground (DHT11 + soil divider)
